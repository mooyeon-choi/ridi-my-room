# 리디 마이룸 (Ridi MyRoom)

> 트래블러스 레스트 스타일의 픽셀 아트로 만드는 나만의 서재
> AI 아바타가 방문자를 응대하는 게이미피케이션 마이페이지

## 📖 프로젝트 소개

리디북스의 독서 활동을 시각화한 인터랙티브 서재 공간입니다.
- 동물의 숲처럼 서재를 자유롭게 꾸미기
- 읽은 책이 책장에 자동으로 쌓임
- AI 아바타가 나를 대신해 방문자와 대화
- 상수리나무 아래 주인공처럼 따뜻한 AI 페르소나

## 🎯 현재 목표: 1주일 프로토타입

**개발 기간**: 2026-04-13 ~ 2026-04-20 (7일)

### 프로토타입 범위

**내 서재 모드 (`/my-room`)**
- ✅ 기본 픽셀 아트 서재 공간
- ✅ 내 아바타 키보드 조종
- ✅ **행동별 AI 혼잣말 생성** (걷기, 책 읽기, 창문 보기 등)
- ✅ **말풍선으로 혼잣말 표시**
- ✅ QR 코드 생성 및 공유
- ❌ 가구 편집 (향후)

**방문 모드 (`/:userId/room`)**
- ✅ **QR 코드로 다른 사용자 초대**
- ✅ 주인 AI 아바타 자율 이동
- ✅ **방문자 아바타 키보드 조종**
- ✅ **AI 채팅 (주인의 GPT API 사용)**
- ✅ "상수리나무" 페르소나 구현

**향후 확장**
- ❌ 실제 리디 API 연동

## 🎭 AI 페르소나: "상수리나무 아래"

### 핵심 특징
- **따뜻함**: 방문자를 진심으로 환대
- **지성**: 책에 대한 깊은 애정
- **서정적**: 문학적이고 은유적 표현
- **정중함**: 존댓말, 부드러운 종결어

### 응답 예시
```
방문자: "안녕"
AI: "안녕하세요. 책 향기 가득한 이 서재에 오신 것을 환영합니다 📚"

방문자: "책 추천해줘"
AI: "당신이 어떤 이야기를 좋아하시는지 궁금하네요.
     감동적인 로맨스, 혹은 모험 가득한 판타지를 원하신다면
     말씀해주세요."
```

**자세한 내용**: `docs/ai-persona-guide.md` 참고

## 🎯 주요 목표

### 프로토타입 성공 기준
1. 서재가 픽셀 아트로 렌더링됨
2. 주인 AI 아바타가 자율적으로 움직임 (걷기 + Idle)
3. **QR 코드 생성 및 공유**
4. **방문자가 QR로 접속하여 아바타 조종 가능**
5. **AI 채팅 작동 (주인의 GPT API 사용)**
6. AI가 "상수리나무" 페르소나로 응답
7. 배포 완료 (프론트엔드 + 백엔드)

### 핵심 사용자 플로우

**내 서재 모드**
```
내 서재 입장 → 아바타 이동 → 행동 (책 읽기, 창문 보기 등)
                                ↓
                          AI 혼잣말 생성
                                ↓
                          말풍선 표시
```

**방문 모드**
```
주인 → QR 생성 → 공유
         ↓
방문자 → QR 스캔 → 서재 입장 → 아바타로 돌아다니기 → AI와 채팅
                      ↑
              주인 AI 아바타가 자율 이동
```

### 장기 비전
- 리디북스 사용자들의 독서 커뮤니티 허브
- 독서량 증가 및 도서 추천 활성화
- 게이미피케이션으로 독서 습관 형성
