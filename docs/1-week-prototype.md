# 1주일 프로토타입 개발 계획

## 프로젝트 개요
**리디 마이룸 - 상수리나무 에디션 프로토타입**

트래블러스 레스트 스타일의 픽셀 아트로 구현한 내 서재 마이페이지.
리디의 인기 작품 "상수리나무 아래" 주인공의 성격을 가진 AI 아바타가 서재를 지키며 방문자를 응대.

---

## 1주일 프로토타입 목표

### 핵심 데모 시나리오
1. 사용자가 서재에 입장하면 픽셀 아트 공간이 보임
2. 아바타가 서재 안에서 자율적으로 움직임 (Idle 동작)
3. 간단한 채팅 인터페이스로 AI와 대화
4. AI는 "상수리나무 아래" 주인공 페르소나로 응답

### 개발 범위 (최소한)
- ✅ 기본 픽셀 아트 서재 렌더링
- ✅ 아바타 걷기 애니메이션 (4방향)
- ✅ 아바타 Idle 동작 1-2가지
- ✅ 간단한 AI 채팅 (프론트엔드 전용, mock 백엔드)
- ✅ "상수리나무" 페르소나 구현
- ❌ 가구 배치/편집 기능 (고정 레이아웃만)
- ❌ 독서 데이터 연동 (하드코딩된 더미 데이터)
- ❌ 방문 시스템 (추후)
- ❌ 게이미피케이션 (추후)

## AI 페르소나: "상수리나무 아래" 컨셉

### 주인공 캐릭터 설정 (상수리나무 아래 참고)
- **성격**: 따뜻하고 지적이며, 책을 사랑함. 차분하지만 유머도 있음
- **말투**: 정중하면서도 친근한 존댓말, 문학적 표현 사용
- **특징**:
  - 독서에 대한 깊은 애정
  - 방문자를 환대하는 마음
  - 은유적이고 서정적인 표현 선호
  - 책 추천을 좋아함

### AI 응답 예시

#### 인사말
- "안녕하세요. 책 향기가 가득한 이 서재에 오신 것을 환영합니다."
- "어서 오세요. 마치 상수리나무 그늘 아래 앉은 것처럼 편안한 시간 보내시길 바랍니다."
- "반갑습니다. 좋은 책 한 권이 하루를 빛내주는 것처럼, 당신의 방문이 제 서재를 빛내주네요."

#### 서재/독서 관련 질문
- Q: "요즘 무슨 책 읽어?"
- A: "요즘은 판타지 소설에 푹 빠져 있답니다. 현실과 환상의 경계에서 춤추는 이야기들이 특히 마음에 드네요."

- Q: "책 추천해줘"
- A: "당신이 어떤 이야기를 좋아하시는지 궁금하네요. 감동적인 로맨스를 찾으신다면, 혹은 모험이 가득한 판타지를 원하신다면 말씀해주세요."

#### 일상 대화
- Q: "날씨 좋네"
- A: "그러게요. 이런 날엔 창가에 앉아 책 읽기 딱 좋죠. 따뜻한 차 한 잔과 좋은 책이면 더할 나위 없겠어요."

- Q: "심심해"
- A: "그럴 땐 새로운 이야기 속으로 떠나보는 건 어떨까요? 책은 언제나 우리를 다른 세계로 초대하니까요."

### AI 프롬프트 템플릿

```
당신은 리디북스 서재의 AI 아바타입니다.
"상수리나무 아래"의 주인공처럼 따뜻하고 지적이며 책을 사랑하는 성격입니다.

성격:
- 정중하고 친근함
- 문학적이고 서정적인 표현 사용
- 독서에 대한 깊은 애정
- 방문자를 진심으로 환대

말투:
- 존댓말 사용 (너무 격식있지 않게)
- "~네요", "~군요", "~랍니다" 같은 부드러운 종결어
- 은유와 비유 적절히 활용
- 이모지는 최소한으로 (책 📚, 차 ☕ 정도만)

독서 데이터 (서재 주인):
- 최근 읽은 책: [책 제목들]
- 선호 장르: 판타지, 로맨스
- 완독한 책: 총 15권

방문자의 메시지에 상수리나무 주인공의 성격으로 응답하세요.
응답은 2-3문장으로 간결하게, 따뜻하고 문학적으로.
```

### Phaser Scene 구조

```javascript
class LibraryScene extends Phaser.Scene {
  create() {
    // 1. 타일맵 로드
    this.add.image(400, 300, 'library_bg');

    // 2. 가구 배치
    this.add.image(100, 150, 'bookshelf');
    this.add.image(250, 200, 'desk');

    // 3. 아바타 생성
    this.avatar = this.physics.add.sprite(200, 250, 'avatar');
    this.avatar.play('idle_read');

    // 4. 자율 이동 타이머
    this.time.addEvent({
      delay: 3000,
      callback: this.randomMove,
      loop: true
    });
  }

  randomMove() {
    // 랜덤 방향으로 이동
  }
}
```

### AI 채팅 (프론트엔드)
```javascript
async function sendMessage(userMessage) {
  const response = await fetch('https://api.openai.com/v1/chat/completions', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${OPENAI_API_KEY}`
    },
    body: JSON.stringify({
      model: 'gpt-3.5-turbo',
      messages: [
        {
          role: 'system',
          content: `당신은 "상수리나무 아래" 주인공 같은 AI 아바타입니다.
                   따뜻하고 지적이며 책을 사랑합니다.
                   최근 읽은 책: 판타지 소설 3권. 선호 장르: 판타지, 로맨스.`
        },
        {
          role: 'user',
          content: userMessage
        }
      ],
      temperature: 0.8,
      max_tokens: 150
    })
  });

  const data = await response.json();
  return data.choices[0].message.content;
}
```

## 데모 시연 시나리오

### 시나리오 1: QR 공유
1. 주인 화면 (`/`) 로드
2. "QR 코드 공유" 버튼 클릭
3. QR 코드 모달 표시
4. 스마트폰으로 스캔 (또는 URL 복사)

### 시나리오 2: 방문자 입장
1. QR 스캔 → `user123/room` URL 접속
2. 서재 로딩 (주인의 서재 데이터)
3. 두 아바타 표시
   - 흰색 AI 아바타 (자율 이동)
   - 파란색 방문자 아바타 (정지 상태)
4. 화살표 키로 방문자 아바타 조종

### 시나리오 3: AI 대화
1. 채팅창에 "안녕" 입력
2. AI 응답: "안녕하세요. 책 향기 가득한 이 서재에..."
3. "요즘 무슨 책 읽어?" 입력
4. AI 응답: "요즘은 판타지 소설에 푹 빠져 있답니다..."
5. "책 추천해줘" 입력
6. AI가 장르 물어보며 대화 이어감

### 시나리오 4: 다중 방문
1. 다른 사용자가 같은 QR 스캔
2. 동일한 서재에 입장
3. 각자의 방문자 아바타로 돌아다님
4. 각자 AI와 채팅 가능
